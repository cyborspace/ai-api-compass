## 📚 术语表

| 术语 | 英文 | 定义 |
|------|------|------|
| **前线部署工程师** | Forward Deployed Engineer (FDE) | 驻场客户公司，负责填补产品功能与客户需求鸿沟的技术工程师 |
| **本体论** | Ontology | 描述现实世界实体及其关系的语义模型，Palantir 平台的核心抽象 |
| **对象类型** | Object Type | Ontology 中定义的业务实体（如用户、工具） |
| **属性** | Property | Object Type 的特征（如名称、评分） |
| **链接类型** | Link Type | Object Type 之间的关系（如用户拥有订阅） |
| **动作类型** | Action Type | 对 Object 的业务操作（如提交评价、创建工具） |
| **语义层** | Semantic Layer | Ontology 的定义层，描述"世界是什么" |
| **动力层** | Kinetic Layer | Ontology 的数据集成层，负责数据映射和同步 |
| **动态层** | Dynamic Layer | Ontology 的业务逻辑层，负责 Actions 和 Functions |
| **AIP** | Artificial Intelligence Platform | Palantir 的 AI 平台，集成 LLM 能力 |
| **AIP Logic** | AIP Logic | Palantir 的 LLM 编排语言，用于构建 AI Agent |
| **Workshop** | Workshop | Palantir 的低代码应用构建工具 |
| **Slate** | Slate | Palantir 的全代码应用构建工具 |
| **Quiver** | Quiver | Palantir 的数据分析和可视化工具 |
| **Foundry** | Foundry | Palantir 的企业级数据操作系统 |
| **数据血缘** | Data Lineage | 数据从源头到目的地的流转路径追踪 |
| **回写** | Writeback | 将 Ontology 中的变更同步回源系统 |
